Routes...

  1. GET / - displays all of the scraped data from the "articles" collection in the newsMongoDB database for end users to read and comment on
  2. GET /scrape - sends a request to scrape a website's html content and store the desired parts in the "articles" collection in the newsMongoDB database
  3. GET /articles - displays ALL of the scraped data from the "articles" collection in the newsMongoDB database in JSON format
  4. GET /articles/id - displays ONE item of the scraped data from the "articles" collection in the newsMongoDB database in JSON format
  5. POST /articles/id - creates a "notes" entry in the newsMongoDB database based on user input and ties it to a specific article in the "articles" collection.
